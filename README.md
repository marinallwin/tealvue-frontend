# TealVue Real-Time Market Dashboard (Frontend)

## Overview

This React application provides a real-time market dashboard that displays live price updates and anomaly alerts generated by the backend service.

The dashboard receives live data through Socket.IO and visualizes market activity using interactive charts.

---

## Features

### Live Price Charts

- Displays configured market symbols
- Updates prices in real time
- Uses Lightweight Charts for visualization
- Handles continuous streaming updates efficiently

### Live Alerts

- Receives anomaly alerts instantly through Socket.IO
- Displays alerts as they are generated
- Shows alert reference, symbol, and reason

### Historical Alerts

- Fetches the latest alerts from the backend API
- Allows switching between live alerts and stored alerts

### Connection Status

Displays the current connection status with the backend:

- Connected
- Disconnected
